# 语音记账App — 开发环境清单

> 逐项检查，完成后在状态栏打勾 ✅

---

## 1. Java 环境

- [ ] JDK 17 已安装（路径：`C:\Program Files\Java\jdk-17`）
- [ ] `JAVA_HOME` 环境变量指向 JDK 17
- [ ] `Path` 中包含 `%JAVA_HOME%\bin`（去掉 JDK 8 的路径）
- [ ] 验证：打开 PowerShell 输入 `java -version`，显示 `17.x.x`

---

## 2. Android SDK

- [ ] Android Studio 已安装
- [ ] Android SDK 路径已配置（如 `C:\Users\你的用户名\AppData\Local\Android\Sdk`）
- [ ] 以下 SDK 组件已安装（通过 Android Studio → Tools → SDK Manager）：
  - [ ] Android SDK Platform（API 34 或最新稳定版）
  - [ ] Android SDK Build-Tools
  - [ ] Android SDK Command-line Tools (latest)
  - [ ] Android Emulator
- [ ] 验证：`flutter doctor` 中 Android toolchain 显示绿色 ✓

---

## 3. Flutter SDK

- [ ] Flutter SDK 已下载并解压（路径建议 `C:\flutter`，不要含空格或中文）
- [ ] `Path` 中添加了 `C:\flutter\bin`
- [ ] 验证：新 PowerShell 窗口输入 `flutter --version`，显示版本号

---

## 4. Android 许可证

- [ ] 已运行 `flutter doctor --android-licenses` 并接受所有许可
- [ ] 验证：`flutter doctor` 无 Android license 报错

---

## 5. VS Code 插件

- [ ] Flutter 插件已安装
- [ ] Dart 插件已安装（装 Flutter 会自动装）
- [ ] 重启 VS Code

---

## 6. 模拟器 / 真机

- [ ] Android 模拟器已创建（Pixel 6 等）或真机已连接并开启开发者模式
- [ ] 验证：`flutter devices` 能看到设备

---

## 7. API 密钥

- [ ] DeepSeek API Key（https://platform.deepseek.com）
- [ ] Whisper API Key 或其他语音转文字服务的 Key（https://platform.openai.com 或其他）

---

## 验证总检

全部完成后运行：

```
flutter doctor
```

预期结果：

```
[✓] Flutter
[✓] Android toolchain
[✓] Android Studio
[✓] VS Code
[✓] Connected device
```
